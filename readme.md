# template-kit

### A simple template generator which can be invoked via the CLI

<br/>

## Installation

```bash
# npm registry
pnpm @mrmihi/template-kit

# using pnpm link
pnpm link --global

# using npm pack
npm pack 
npm install --global {tarball}
```

## Usage

- Execute the following command in your terminal to see a list of available options

```bash
sft
```
